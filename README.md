#### Обо мне
Работаю тестировщиком уже 2 года. Тестирую в основном сайты, а именно интернет-магазины, также участвовал в тестировании мобильного приложения где тестировал REST API с помощью инструмента Postman.

#### Навыки
- Уверенное использование в работе техник тест-дизайн
- Создание и поддержка тест-кейсов и чек-листов
- Работа с баг-трекерами (Test IT, DoQa, Jira, TestRail)
- Основы SQL, REST API и SOAP API тестирования 
- Базовые знания HTML, CSS, Git
- Создание автотестов с помощью PlayWright и Pytest на Python
- Создание сценариев для Apache Jmeter и проведение нагрузочного тестирования
